# Manejo de Promesas en JavaScript

Ejercicio práctico con 3 promesas que simulan peticiones a BD, verifican la hora del sistema y ejecutan procesos pesados.

## Las 3 Promesas

1. **Promesa 1** - Simula una petición a BD con setTimeout, retorna un objeto Persona
2. **Promesa 2** - Verifica los segundos actuales y retorna la hora o un error según sea PAR o IMPAR
3. **Promesa 3** - Ejecuta un proceso pesado con un bucle grande para observar el delay

## Archivos

- `index.html` - Interfaz
- `style.css` - Estilos
- `script.js` - Las 3 promesas
